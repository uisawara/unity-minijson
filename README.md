# unity-minijson
MiniJSON for Unity PackageManager

# Target

* upper Unity 2018
* Require Unity PackageManager

# installation

* Add lines to Packages/manifest.json

```
{
  "dependencies": {
    "work.mmzk.libs.minijson": "https://github.com/uisawara/unity-minijson.git"
  }
}
```
